Never use LaTeX math notation. Always use Typst syntax.

**Why:** User cannot read LaTeX; this project uses Typst for the thesis.
**How to apply:** Use `$...$` for inline math, Typst symbols (`dot.circle`, `RR`, `CC`, `bold(1)`, `exp`, `arg`, `abs(...)`, `sum`, `prod`, etc.) everywhere math appears in responses.
